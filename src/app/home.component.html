Demonstrates https://github.com/angular/angularfire2/issues/574

Reproduction steps

  • Add firebase credentials to app.module.ts
  • Click "Generate items" to get some example data (do this only once)
  • Refresh the page
  • Open console
  • Notice "Got items" outputted once with a single array of objects
  • Click "find items of type 1"
  • Notice "Found items of type 1" is printed multiple times as opposed to just once as expected (see example below)
    Found items of type 1:  []
    Found items of type 1:  [Object]
    Found items of type 1:  [Object, Object]
    Found items of type 1:  [Object, Object, Object]
    Found items of type 1:  [Object, Object, Object, Object]
    Found items of type 1:  [Object, Object, Object, Object, Object]
    Found items of type 1:  [Object, Object, Object, Object, Object, Object]
